# Jubilee Superstore Financial Performance Dashboard

## Project Overview 

* **Financial Performance Tracking (Part A):** Evaluates and analyzes financial performance data against corporate strategic targets (specifically focusing on Gross Profit Margin and Operating Profit Margin). The purpose is to design an interactive dashboard that visualizes data across multiple dimensions including product category, geographic region, customer type, division, and distribution channel to support management in data-driven decision making.
  
* **Balanced Scorecard Development (Part B):** Suggests KPIs to develop the company's Balanced Scorecard by outlining actionable metrics to monitor and improve performance in relation to customer satisfaction, internal process efficiency, workforce learning, social responsibility, environmental sustainability, and Indigenous data sovereignty.
